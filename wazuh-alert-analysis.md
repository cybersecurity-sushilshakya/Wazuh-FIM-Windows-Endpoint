# Wazuh Alert Analysis

Each FIM alert generated by Wazuh included the following details:
- File path
- Event type (added, modified, deleted)
- User account involved
- Timestamp
- File hash values

---

## Alert Severity
- File creation alerts were classified as medium severity.
- Modification and delete of files under directories was classified as high severity.

---

## Analyst Observations
Executable creation in non-standard directories is a common indicator of malware or unauthorized activity and requires immediate investigation.
