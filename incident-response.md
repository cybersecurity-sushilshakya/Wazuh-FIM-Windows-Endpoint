# Incident Response Process

## Detection
File Integrity Monitoring (FIM) alerts were generated by Wazuh SIEM.

## Analysis
The file path, user context, and change type were reviewed.

## Classification
The activity was classified as unauthorized file manipulation.

## Containment
Endpoint isolation was simulated as a containment measure.

## Remediation
Unauthorized files were removed from the endpoint.

## Documentation
The incident was documented for audit and compliance purposes.
